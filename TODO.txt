Steps to apply a repo to github

1-git init
2-git add .
3-git commit -m "test"
4-connect remote repo to git. (git remote add origin https://github.com/yourusername/your-repo.git)
5-Git remote -v (check remote connection data like above.)
6-git branch (check branch name)
7-git push --set-upstream main main (it will set main as upstream branch and push it to git.)
